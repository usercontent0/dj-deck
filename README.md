# dj-deck
This is the codebase for all of my deck models. It is structured as a set of **templates** (deck + mixer control scripts / modules) plus a small init module that clones the right templates into the deck model at runtime.

## Supported models
### Mixers
- Pioneer DJ DJM-900NXS2
- Pioneer DJ DJM-V10
- Pioneer DJ DJM-A9
- Allen & Heath Xone:96

### Players
- Pioneer DJ CDJ-2000NXS2 
- Pioneer DJ CDJ-3000 

## Note
This repo contains ONLY the scripts; it does not contain any models or textures. You are free to use this code for your own decks as long as you comply with this repo's license. <br>

Issues and PRs are welcome incase you find any bugs or would like a feature added.
